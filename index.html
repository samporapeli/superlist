Keys:

  • Arrow keys = Look around
    • Up = Next item
    • Down = Previous item
    • Right = Subtasks
    • Left = Bigger picutre
  • C/D/T/Q/W/X/F = Set item state
    • C = "checkbox" = do this
    • D = done
    • T = text description
    • Q = idea / can be done but not needed
    • W = waiting / needs to be done but cannot be worked on yet
    • X = cancelled / not to be done anymore
    • F = tried doing but failed
  • C/D/T/Q/W/X/F + Arrow keys = Create new item
  • Backspace = Delete item
  • Cmd+alt + Arrow keys = Move active item around
  • Return = Edit item text or save edited text
  • S = Set schedule (deadline) for item. Day month year hour.
    • Schedule propagates left unless the item is Done or Cancelled
